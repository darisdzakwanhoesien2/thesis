# Japfa Sustainalytics

Kami menggunakan platform internal bernama Japfa Sustainalytics untuk menganalisis informasi yang terhimpun melalui JSRS, termasuk data mengenai karyawan. Tahun ini, kami mengimplementasikan penggunaannya di berbagai negara tempat Japfa operasi dan berfokus pada input data untuk menjamin ketepatannya. [GRI 401-1]

We use an internal platform called Japfa Sustainalytics to analyse insights gathered through JSRS, including data on employees. This year, we implemented its use in the various countries where Japfa operates and focused on data input to ensure accuracy. [GRI 401-1]
